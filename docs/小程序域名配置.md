# 小程序服务器域名配置

## 需要配置的域名

在微信小程序后台配置以下域名，才能正常使用地图功能：

### 1. request合法域名（必须）
```
https://apis.map.qq.com
```

### 2. downloadFile合法域名（可选）
```
https://apis.map.qq.com
```

## 配置步骤

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入你的小程序后台
3. 左侧菜单「开发」→「开发管理」→「开发设置」
4. 找到「服务器域名」部分
5. 点击「修改」，扫码验证
6. 在对应位置添加上述域名
7. 保存

## 开发者工具设置

在开发阶段，可以在开发者工具中跳过域名校验：

1. 点击右上角「详情」
2. 找到「本地设置」
3. 勾选「不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书」

⚠️ **注意**：发布前必须配置真实域名，此选项仅用于开发调试！

## 配置完成后

1. 重启微信开发者工具
2. 重新编译小程序
3. 测试定位功能
4. 测试逆地理编码功能（地址解析）

## 腾讯地图SDK相关说明

- SDK文件路径：`miniprogram/libs/qqmap-wx-jssdk.min.js`
- 腾讯地图API基础地址：`https://apis.map.qq.com/ws/`
- 需要申请腾讯地图Key并在代码中配置
